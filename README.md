# Students Performance Dataset Analysis

## Objective
To understand the structure, feature types, and machine learning readiness of the
Students Performance Dataset.

## Dataset
students_performance_data.csv

## Tasks Performed
- Dataset loading and inspection
- Manual feature type identification
- Target variable identification
- Class imbalance analysis
- Data quality observations

## Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook

## Outcome
The dataset was found to be suitable for supervised machine learning tasks after
appropriate preprocessing.
